# OYED Dotfiles

## Applications

- i3-gaps - Window Manager
- pywal - Theming
- picom - Compositor
- rxvt-unicode - Terminal
- vscode - IDE
- polybar - Persistent Taskbar
- firiefox-developer-edition - Web Browser
- trizen - AUR Helper
- autorandr - Multi-monitor support
- cava - Audio Visualizer
